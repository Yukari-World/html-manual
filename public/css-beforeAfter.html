    疑似要素 :before / :after

    疑似要素 :before / :afterとは

    ある要素の前、もしくはその後に追加される疑似要素である。指定がなければ中身は常に空であり、ブラウザ上にある検証機能にも表示されない。

    使用方法と解説

    SCSS

    ul {
	margin-left: 1em;
	padding-right: 2px;
	text-indent: -1em;

	li {
		&::before {
			content: "├";
			animation: flashString 10s linear infinite;
		}

		&:last-child {
			&::before {
				content: "└";
				animation: flashString 10s linear infinite;
			}
		}
	}
}

    CSS

    ul {
	margin-left: 1em;
	padding-right: 2px;
	text-indent: -1em;
}

ul li::before {
	content: "├";
	animation: flashString 10s linear infinite;
}

ul li:last-child::before {
	content: "└";
	animation: flashString 10s linear infinite;
}

    出力結果はメニューリスト通りなので省略する。
    リストタグ<li>に対して利用する例。リストの前に追加することでツリー構造のような見た目を作ることができる。擬似要素:last-childを使うことでツリー構造の終点を別のものに置き換えている。

    afterの方は説明例が思いつかなかったため現在は省略。

    使用上の注意

    • before / afterの要素を利用する場合、content要素を必ず必要とする。空の文字列を用意するだけでも問題はない。
    • 疑似要素で追加された文字列はマウスドラッグによって対象にすることができない。またコピーの対象にもできない。
