      グラデーション

      HTML上でのグラデーションはいくつか方法があるがその一例

      使用方法と解説

      background: linear-gradient(to bottom, rgba(0, 0, 0, 0) 0%, rgba(0, 0, 0, 0.6) 100%);

      dyadで使用されている直線グラデーション。
      下から上に半透明の黒のグラデーションを行う。
      なお、backgroundプロパティは backgroundの各プロパティのショートハンドとして補完しており、background-imageに対して適応する。

      background: linear-gradient(45deg, red, blue);

      Mozillaに公開されている例。対角線上に赤から青となるグラデーションが生成される。

      background: linear-gradient(to top, #004B24 0%, #DBE5E0 86.45%, #FFFFFF 100%);

      三色グラデーションの例。カンマで繋ぐ事で複数色設定できる。更に割合を指定することで色の変わり目の位置を指定することができる。

      background: linear-gradient(to right, #000011, #1164FF, #FFFF99);

      本マニュアルで使用されているグラデーション。%表記が省略された場合は色の切れ目が均等な位置になるように位置が割り振られる。左側の色は背景色と同じ色のため殆ど見えない。尚、このマニュアルの背景は黒ではない。

      使用上の注意

      • これらのCSSはブラウザ間の互換性が低い点に注意が必要。特にIEや古いスマートフォンを利用しているユーザーからは見えない可能性もある。対処としてはこれとは別に背景画像を用意することで完全ではないものの回避できる。
      • グラデーションは画像扱いであるため、取扱の間違いには注意。colorではなく、imageなので要注意。
      • 複数利用する場合、スペックによっては表示速度に著しく影響するので注意が必要。対処法としてグラデーション背景を用意する方法がある。
      • 画像タグ(<img>)に対して適応する場合、beforeに付与する必要がある。また、必要に応じてz-indexによる重なりを気にする必要がある。
      • 複数のブラウザで実装する場合、ベンダープレフィックスを忘れずに。gulpで自動付与する方法もあるが、詳細はgulpの項目を参照。
