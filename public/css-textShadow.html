文字装飾による可読性の向上

説明

画像の上に文字を出力させる時、白地に白文字や黒地に黒文字といったことが発生することがある。これは可読性に大きく影響する。しかし、文字装飾を付与することで可読性の向上を望める場合がある。

使用方法と解説

text-shadow: 0 0 6px #000000;

I was once an adventurer like you;then I took an arrow in the knee.

文字の真下にぼかし6pxの黒の影を付与する。数値が小さい程文字の原型が残り、大きくなる程ぼやけたようになる。

text-shadow: 5px 5px 2px #0000FF, -5px -5px 1px #FF0000;

I was once an adventurer like you;then I took an arrow in the knee.

赤と青の影を付与する。カンマを使用することで複数定義することができる。数の制限は特にないが、あくまで装飾に過ぎず、多く指定し過ぎると却って可読性を損なう要因となる。

使用上の注意

  • 文字色と背景に合わせて文字装飾の色を調整することを忘れずに。
