    input要素

    書きかけのページ

    このページの内容は書きかけです。不定期なタイミングで記述内容が追加、変更されることがあります。

    説明

    <input>要素は<form>要素と一緒に使われることの多いホームページを訪れたユーザーがデータを入力する要素の一つである(ユーザーが入力するその他の要素は<textarea>、<button>等がある)。<input>は非常に種類が多く、利用頻度が高いがその全てを把握するのが難しいのでこのページで一通り纏めておく。

    inputの種類

    button

    機能を持たないボタンを作成する。

    <input type="button" value="ボタン">

    checkbox

    チェックボックスを作成する。ラジオボタンと異なり複数選択することができる。

    <label><input type="checkbox" value="1">内容1</label>
<label><input type="checkbox" value="2">内容2</label>
<label><input type="checkbox" value="3">内容3</label>

    color

    <input type="color" value="#FFFFFF">

    date

    日付入力フォームを作成する。ブラウザによってはカレンダーが表示されたりする。

    <input type="date">

    datetime-local

    <input type="datetime-local">

    email

    メールアドレス入力フォームを作成する。ブラウザ毎にメールアドレスの検証機能が異なる。

    <input type="email">

    file

    <input type="file">

    hidden

    <input type="hidden">

    image

    <input type="image">

    month

    <input type="month">

    number

    <input type="number">

    password

    パスワード入力フォームを作成する。パスワード入力時、文字は全て●で表示される。また、非SSLページ(暗号化されていないページ)ではブラウザによっては警告が表示される。

    <input type="password">

    radio

    複数の選択肢から1つを選ぶラジオボタンを作成する。

    <label><input type="radio" value="1">内容1</label>

    ラジオボタンはname属性に同じ名前を指定することで同一グループであると認識する。

    <label><input type="radio" name="bread" value="1">内容1</label>
<label><input type="radio" name="bread" value="2">内容1</label>
<label><input type="radio" name="bread" value="3">内容1</label>

    range

    数値の正確さが重要ではない数値の入力要素を作成する。

    <input type="range">

    reset

    入力されている項目を初期化する

    <input type="reset">

    search

    <input type="search">

    submit

    フォームの入力を完了し送信するボタンを作成する。

    <input type="submit">

    tel

    電話番号を入力フォームを作成する。

    <input type="tel">

    text

    <input type="text">

    time

    <input type="time">

    url

    <input type="url">

    week

    <input type="week">

    使用可能な属性

    autofocus

    そのページに移行した時、自動的にそのフォームに入力カーソルが置かれる。autofocusには内容が存在しないためautofocusのみでも問題はない。尚サンプルは性質上ページこの位置まで勝手に移動してしまうので用意していない。

    <input type="text" name="your-name" autofocus="autofocus">

    formaction

    フォームに入力された情報を処理するプログラムのURIを指定する。<form>要素に指定したaction属性よりも優先される。この属性はimageもしくはsubmitにのみ指定できる。

    <input type="submit" formaction="sample.php">

    list

    name

    フォーム項目の名前。ブラウザ上には表示されないため<label>等を使用してどの様な項目なのか記入する必要がある。JavaScriptのパーツとして使用する場合を除きほぼ必須項目である。

    <input type="text" name="your-name">

    required

    そのフォームの入力を必須にする。requiredには内容が存在しないためrequiredのみでも問題はない。 checkboxに使用した場合、必ずその項目をチェックをつけなくてはならない。

    value

    checkbox、hidden、submit、radioとそれ以外では扱いが異なる。

    • checkboxとradioの場合
      選択した時に送られる値。
    • hiddenの場合
      フォームに送られる値。JavaScript等使わない限り編集することができない。
    • submitの場合
      ボタンに表示される文字列。フォーム送信時にその文字列も一緒に送られる。
      <input type="submit" name="submit" value="送信確認">
    • それ以外の項目の場合
      フォーム項目の初期値。
      <input type="text" name="your-name" value="山田太郎">

    使用上の注意

    • <form>要素上に存在しない入力フォームは基本的に送信されない。formから独立した位置に入力フォームを設置する場合はform属性を忘れないようにすること。但し大抵これを使用する場合入力フォームの位置が通常より遠ざかる位置にあることが多いためフォームの形としては不適切になりやすいことには注意すること。またHTML5の要素故に旧バージョンのブラウザでは対応していないことが多いので注意が必要。
