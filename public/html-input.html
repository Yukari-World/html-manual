      input要素

      書きかけのページ

      このページの内容は書きかけです。不定期なタイミングで記述内容が追加、変更されることがあります。

      説明

      input要素はform要素と一緒に使われることの多いホームページを訪れたユーザーがデータを入力する要素の一つである(ユーザーが入力するその他の要素はtextarea、button等がある)。inputは非常に種類が多く、利用頻度が高いがその全てを把握するのが難しいのでこのページで一通り纏めておく。

      button

      <input type="button" value="ボタン">

      checkbox

      <label><input type="checkbox" value="1">内容1</label>

      color

      <input type="color" value="#FFFFFF">

      date
