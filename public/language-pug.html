    Pug

    Pugとは

    PugはHTMLのテンプレートエンジンである。かつてはJadeと呼ばれていたが、商標権との関係でPugと呼ばれている。Jade自体は現在でも存在し、コンパイラも存在するが、非推奨となっている。
    尚、PugもJadeも元は同じ為記述方法自体は変わらないが、基本的には現行環境であるPugを基に記述していく。

    記述方法

    基本的な記述

    記述方法はRubyスタイルに近い。タブで階層を示し、HTMLタグを囲む必要はない。但し、HTMLタグをそのまま記述しても特に問題はない。

    Pug

    doctype html
html(lang='ja')
	head
		meta(charset='utf-8')
		title Debug Checker
		script(type='application/javascript', src='js/prism.js')
		link(href='css/prism.css', rel='stylesheet', type='text/css')
		link(href='css/style.css', rel='stylesheet', type='text/css')
	body
		header(role='banner')
			h1 Title
		main(role='main')
			h2 Hello World!!
		footer(role='contentinfo')
		script(type='application/javascript', src='js/prism.js')

    HTML

    出力結果。実際の出力結果は全く整形されていないので必要に応じてJS-Beautify等で整形する必要がある。尚、Pugの変換オプションに自動整形するオプションがあるが、現在は非推奨なので注意。

    <!DOCTYPE html>
<html lang="ja">
<head>
	<meta charset="utf-8">
	<title>Debug Checker</title>
	<link href="css/prism.css" rel="stylesheet" type="text/css">
	<link href="css/style.css" rel="stylesheet" type="text/css">
</head>
<body>
	<header role="banner">
		<h1>Title</h1>
	</header>
	<main role="main">
		<h2>Hello World!!</h2>
	</main>
	<footer role="contentinfo"></footer>
	<script type="application/javascript" src="js/prism.js"></script>
</body>
</html>

    プロパティ

    プロパティは括弧で記述する。複数プロパティがある場合はカンマで区切る。但し、こちらは必須ではない。

    Pug

    input(type='text', name='school', placeholder='学校名')

    HTML

    <input type="text" name="school" placeholder="学校名">

    複数行の記述

    <p>タグの様に内容が長くなる場合や同一タグの中に文章とHTMLタグが混在する場合、文章部分は|で始める。

    Pug

    p
	| 参照元は
	a(href='#', target='_blank') こちら

    HTML

    <p>参照元は<a href="#" target="_blank">こちら</a></p>

    尚、どうしても一行で済ましたいならば以下のように記述する。

    Pug

    p 参照元は<a href="#" target="_blank">こちら</a>

    IDとクラス

    IDとクラスはプロパティを記述する方法で記述してもよいが、以下のように記述することができる。

    Pug

    table.contets
	tr
		td#test1 Test1
		td#test2 Test2

    HTML

    <table class="contets">
	<tr>
		<td id="test1">Test1</td>
		<td id="test2">Test2</td>
	</tr>
</table>

    また、<div>タグにIDやクラスを記述する場合、<div>タグを省略して記述することができる。

    Pug

    #top.box
	p Message
#bottom.box
	p Message

    HTML

    <div id="top" class="box">
	<p>Message</p>
</div>
<div id="bottom" class="box">
	<p>Message</p>
</div>

    同一のHTMLタグに複数のクラスを指定する場合は、CSSの記述方法の様に連続して記述する。

    Pug

    .box.left
	p.this.is.four.pens Test

    HTML

    <div class="box left">
	<p class="this is four pens">Test</p>
</div>

    include インクルード

    Pug

    複数のパーツを使い回したり、パーツを細分化する際に使用する。尚、pugテンプレートファイルを指定する場合は拡張子を記述する必要はない。

    
				
    extends 継承
    
				
    Pug
    
				
    親ファイル。blockを追加することで継承先にて要素を追加、もしくは置き換えることができる。要素に何も追加しない場合、出力時も何も出力されない。
    
				
    子ファイル。extendsで継承元のテンプレートファイルを指定する。ファイル先頭コメントを除いて必ず1行目に指定しなければならない(1行目以外指定した場合エラーとなる)。
    blockで継承元のブロック名を指定することでブロックの内容を置き換えることができる。
    
					block appendもしくはappendでブロック要素を指定した場合、継承元の内容を保持しつつ要素を追加することができる。blockを指定していない箇所がある場合、指定されていない内容は継承元の内容が出力される。
    
				
    
			
    
			
    
				
    変換方法
    
				
    Pugはテンプレートエンジンであるため、そのままではHTMLとして認識されない。そのため変換する必要がある。これはその一例である。
    尚、此等を利用するにあたって予めNode.jsのインストールが必要となる。インストールしていない場合は別とマニュアルを参照してインストールを行うこと。
    また、内容に応じてプロジェクトパッケージを必要とするためそちらも予め作成しておくように。
    
				
    コマンドラインで変換
    
				
      
					
    1. 
						
      コンソールを起動し、以下のコマンドからPugのインストールを行う
      npm i pug-cli -g
      
						
      ローカルパッケージにインストールする場合は以下のように入力を行う。ローカル環境インストールの場合、プロジェクトパッケージが存在するディレクトリに予め移動すること。
      npm i pug-cli -D
      2. 
					
    2. 
						
      インストールに完了した場合、正しく動作するか確認するために以下のコマンドを入力する。
      pug --version
      
						
      ローカルパッケージにインストールした場合は以下のように入力を行う。
      npx pug --version
      3. 
					
    3. 
						
      ファイルを変換する場合、以下のように入力する。ファイル名は変換する対象ファイルに応じて適時変更すること。
      pug index.pug
      
						
      ローカルパッケージにインストールした場合は以下のように入力を行う。
      npx pug index.pug

      4. 
				
    
				
    Gulpで変換
    
				
    予めGulpをインストールすること。インストール方法と使用方法は別マニュアル参照。
    
				
      
					
    1. 
						
      プロジェクトパッケージにがあるフォルダに移動し、以下のコマンドを入力を行う。
      npm i gulp-pug -D
      2. 
					
    2. 
						
      gulpfile.jsが存在しない場合は作成を行い、以下のように入力を行う。ファイル行頭が_(アンダーバー)のファイルを除外することで各種テンプレートファイルを変換しなくなる。
      const pug = require('gulp-pug');

gulp.task('pug', function {
	return gulp.src(['pug/**/*.pug', '!pug/**/_*.pug'])
	.pipe(pug())
	.pipe(gulp.dest('/../'));
});

      3. 
					
    3. 
						
      保存した後に以下のコマンドを入力し、実行する。
      gulp pug
      
						
      gulpをローカルパッケージにインストールした場合は以下のように入力を行う。
      npx gulp pug

      4.
