[MySQL] UPDATE SELECT

説明

SQLデータベースには『方言』が存在する。この『方言』の存在によって通常とは異なる方法で行う必要が出てくることがある。これはその『方言』の一つで、SQL ServerではUPDATE SELECTで行えることがMySQLでは行えない。しかし、以下の方法でそれと同じことができる。

使用方法と解説

以下の方法で同じことを再現できる。

UPDATE
	`table1`,
	`table2`
SET
	`table1`.`value` = `table2`.`value`
WHERE
	`table1`.`id` = `table2`.`id`
;

使用上の注意

  • UPDATE句なのでテストを行う際はTRANSACTION句と併用することを推奨する。TRANSACTION句の使用方法は既に解説があるので省略する。
